"# Voice-Command-Calculator-Assistant" 

A voice-controlled calculator that understands spoken math and commands using speech recognition, with a clean GUI showing waveform visualizations.

**Features:**
- Real-time microphone input with waveform display
- Speech-to-text for math expressions ("what is twenty three times forty seven")
- NLP parser that converts spoken words to mathematical operations
- Support for complex expressions, unit conversions, variable storage
- Voice feedback using text-to-speech
- Command history with editable log

**Skills to Learn:**
- Audio signal processing (waveforms, sampling rates, FFT)
- Speech recognition pipeline understanding
- Natural language to structured data conversion
- Text-to-speech synthesis

**Tools:**
- `SpeechRecognition` — microphone speech-to-text
- `pyttsx3` — offline text-to-speech (no API needed)
- `numpy` — audio signal processing
- `matplotlib` — waveform visualization
- `tkinter` — GUI interface
- `re` — expression parsing
